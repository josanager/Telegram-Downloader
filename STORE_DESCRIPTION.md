# Telegram Video Downloader Pro - Chrome Web Store Listing

## Short Description
High-speed media downloader for Telegram Web (K and Z). Save protected videos and images directly to your Downloads folder with zero quality loss.

## Detailed Description
The ultimate companion for Telegram Web users. **Telegram Video Downloader Pro** allows you to save any video or image from the Telegram Web K and Z interfaces, even from groups or channels where saving/forwarding is restricted.

### Key Features:
- **Byte-Relay Technology**: Our advanced engine captures video data piece by piece to bypass "404 Not Found" errors common in other downloaders.
- **Native Filename Enforcement**: No more random IDs. Files are saved as 'telegram_video_[timestamp].mp4' automatically.
- **Zero Quality Loss**: Captured bytes are 100% identical to the original source. No re-compression or screen recording.
- **Smart UI Integration**: Adds a discrete 'Download' button to video thumbnails and a professional tool in the Media Viewer header.
- **Real-time Progress**: Monitor your download status with a sleek, minimalist progress bar.

### How to use:
1. Open Telegram Web (K or Z versions).
2. Hover over any video to see the red download button.
3. Or open a video in the viewer to see the Blue 'Save' button in the header.
4. Click and wait for the "Byte-Relay" to finish. Your file will appear in your system's 'Downloads' folder.

### Privacy First:
This extension runs entirely locally. It does NOT collect, store, or transmit any of your personal data, chat history, or login credentials. All data processing happens within your browser.
